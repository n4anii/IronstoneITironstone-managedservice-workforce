This PowerShell script combines all the Autopilot Hash to a single csv file. 

1. Dump all the csv files into the "Files" folder
2. Start PowerShell As Admin and go to the directory where the this script is (example C:\temp) and run the CombineAutopilotHashes.ps1.

example:


In powershell

cd C:\temp
.\CombineAutopilotHashes.ps1
